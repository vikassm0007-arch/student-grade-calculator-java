# 🎓 Student Grade Calculator (Java)

## 📌 Overview
The Student Grade Calculator is a simple Java program that allows users to input multiple grades and calculates the average grade efficiently.

## 🚀 Features
- Accepts user-defined number of grades
- Uses arrays to store input values
- Calculates average using basic arithmetic operations
- Handles invalid input (e.g., zero or negative entries)

## 🛠️ Technologies Used
- Java
- Scanner Class
- Arrays & Loops

## 📥 How It Works
1. User enters the number of grades
2. Inputs each grade one by one
3. Program calculates total and average
4. Displays the final average grade
